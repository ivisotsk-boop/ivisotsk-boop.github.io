Website Evaluations

Red Ventures

This website belongs to a large web-development company called Red Ventures

  • Contrast: Website uses good contrast. There are multiple colors and even animation that are easy to read.

  • Repetition: Website uses similar size fonts and bold styles for similar sections. Similar theme colors used throughout the page.

  • Alignment: Small explanatory widgets are aligned with text on one side and an animation on the other. These animations/text segments alternate sides but keep a clean look.

  • Proximity: Spaces between different segments and borders from the edge of the screen to the text make this website easy to read. Text is large enough that only one segment of text is focused on the screen at a time.

Berkshire Hathaway

This website belongs to a conglamorate company owned by Warren Buffet

  • Contrast: Text is readable but better colors could be used. Default purple color on links does not look good.

  • Repetition: Very simple page not much room for repetition but there is a bulleted list that uses the same style for each element. Same style colors used throughout the page.

  • Alignment: Borders are inconsistent on the page forcing the user to jump from left to right to center. Bulleted list consists of two columns with too much whitespace.

  • Proximity: Proximity is not too bad. Less borders could be used in-between sections or more space between borders and text.
